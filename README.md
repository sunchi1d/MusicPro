# 使用说明
## 实现的功能
1、音乐列表、歌单、歌手、排行、榜单、推荐
2、音乐播放、暂停、上一曲、下一曲、喜欢
3、播放列表、添加到播放列表、历史列表
4、搜索单曲、歌手、专辑、MV
5、查看歌手页面、专辑页面、MV
6、热门搜索
7、搜索历史记录
8、排行榜
9、切换播放模式
10、歌词
11、个人中心、本地喜欢、网盘收藏、历史播放记录
12、个人收藏或创建歌单、推荐歌单
13、新歌速递
14、听歌识曲
